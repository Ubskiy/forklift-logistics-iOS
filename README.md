# Forklift Logistics iOS

iOS-приложение для логистического планирования работы погрузчиков в производственном цехе. Приложение позволяет сравнивать две стратегии управления: жадный алгоритм и имитацию отжига.

## Описание

Приложение предназначено для диспетчеров производственных цехов, где используется цепочка перемещения грузов через несколько станций (S→C1→C2→C3→C4→P). Пользователь может настраивать параметры смены, план отгрузки, начальные остатки и сравнивать эффективность разных стратегий управления погрузчиками.

## Основные функции

- **Диспетчерский расчёт**: сравнение жадной стратегии и имитации отжига
- **Настройка сценария**: параметры смены, план отгрузки, начальные остатки, число итераций
- **Таймлайн**: визуализация занятости погрузчиков во времени
- **Журнал рейсов**: детальная информация о каждом рейсе
- **Статистика маршрутов**: анализ эффективности каждого плеча цепочки
- **Справка**: встроенная документация по алгоритмам и метрикам

## Архитектура

Проект построен с использованием Clean Architecture:

```
├── Core/              # Общая функциональность
│   ├── DesignSystem/  # UI компоненты и стили
│   ├── Networking/    # HTTP клиент и API endpoints
│   └── Utilities/     # Константы, форматирование
├── Data/              # Слой данных
│   ├── DTO/           # Data Transfer Objects
│   ├── Mappers/       # Маппинг DTO → Entities
│   └── Repositories/  # Реализация репозиториев
├── Domain/            # Бизнес-логика
│   ├── Entities/      # Доменные сущности
│   ├── Protocols/     # Протоколы репозиториев
│   └── UseCases/      # Use cases
└── Features/          # UI функции
    ├── Dashboard/     # Главный экран
    ├── Timeline/      # Таймлайн погрузчиков
    ├── Trips/         # Журнал рейсов
    ├── Routes/        # Статистика маршрутов
    ├── ScenarioSetup/ # Настройка сценария
    └── Help/          # Справка
```

## Установка

### Требования

- Xcode 15.0+
- iOS 17.0+
- Swift 5.9+

### Запуск проекта

1. Клонируйте репозиторий:
```bash
git clone https://github.com/your-username/forklift-logistics-iOS.git
cd forklift-logistics-iOS
```

2. Откройте `forklift-logistics-iOS.xcodeproj` в Xcode

3. Выберите целевое устройство (iOS Simulator или физическое устройство)

4. Нажмите ⌘R для запуска проекта

## API

Приложение взаимодействует с backend API:
